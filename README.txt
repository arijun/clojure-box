Clojure Box, an all-in-one Clojure environment


Components
----------

Clojure Box consists of these components:

- Clojure: http://clojure.org
- Clojure Contrib: http://sourceforge.net/projects/clojure-contrib/
- Emacs, via EmacsW32: http://ourcomments.org/Emacs/EmacsW32.html
- clojure-mode: http://github.com/jochu/clojure-mode
- swank-clojure: http://github.com/jochu/swank-clojure
- Slime: http://common-lisp.net/project/slime/

Thanks to all the developers of these excellent packages!


Inno Setup Source
-----------------

The installer is built with Inno Setup v5.2.3. The source file is included
with the distribution. You can fork the source from
http://bitbucket.org/shoover/clojure-box-setup.
